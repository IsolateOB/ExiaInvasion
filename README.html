ExiaInvasion

中文 English

ExiaInvasion 是一个从 blablalink 获取个人账号中的Nikke人物数据并生成练度表的开源爬虫程序。

示例输出

注意事项

  • 需要 Edge 或 Chrome 浏览器。
  • 目前仅能输出简体中文和英文表格。

用法

解压压缩包，浏览器进入 chrome://extensions/ 或 edge://extensions/ 页，启用 开发者模式，点击 加载已解压的扩展程序，选择解压后的文件夹。

ExiaInvasion is an open source crawler program that obtains Nikke character data in personal accounts from blablalink and generates progress tracker.

Example Output

Notice

  • Edge or Chrome is required to run the program.
  • Currently only tables in Simplified Chinese and English can be output.

Usage

Unzip the package. In your browser, go to chrome://extensions/ or edge://extensions/, enable Developer mode, click Load unpacked extension, and select the extracted folder.

Main Page

CRAWLER

MANAGE ACCOUNTS & NIKKES
Click to enter the Management Page to configure accounts and character information.
Merge and Save as ZIP
Will merge all files into a ZIP archive for download after Run is completed.
Save Cookie During Runtime
Will automatically save the account's cookie while running, so the next run can skip the login step.
Export JSON
Will output the raw account data for tabulation alongside the table after Run is completed.
Activate Tab During Runtime
Will switch to the script-operated tab when logging in with account and password. This is mainly for checking errors and handling manual human verification.
SAVE CURRENT ACCOUNT COOKIE
Will save the current browser's login cookie for blablalink.

Merge

After SELECT EXCEL FILE, clicking START MERGING will vertically merge these tables.

Management Page

  • Enter and save Email and Password.
  • When email, password, and cookie are all present, the cookie will be used by default.
  • When the Enable switch is turned on, Run will fetch data for that account row.
